# 手机端子弹屯仓、卡邮件等教程

## 一、 启动界面

![启动界面](/images/image_49.png)

* **网络连接请求** ：这是卡邮件需要的，必须点击确定。
1. 界面上其他的菜单功能就不多说了，这里可以看[电脑端的登陆界面功能介绍](/Install/Login)。是一样的

## 二、 主界面

![主界面](/images/image_50.png)

* **左上角**： 打开侧面栏。
* **右上角**： 打开软件设置。
* **入手价格** ：比如交易行稳定价格是638，我们可以填写一个扣除手续费之后的价格。
* **购买次数**： 屯子弹设置为0，屯配件、装备、房卡都需要设置指定数量。(目前还未实现购买指定次数后停止)
* **挡屎价格** ：设置为入手价格的一半。
* **是否有兑换**： 交易行点击要屯的类型子弹购买界面。右下角会显示兑换选项。如果有就勾上，没有就不勾。
* **屯仓-卡邮件-屯仓后卡邮件** ：前面2个选项就不用多说了，字面意思，第三个选项是屯仓到指定次数后会卡邮件，次数在右上角设置里填写。（卡邮件暂未支持）
* **配装方案**： 可以配置4个子弹的方案。
* **切配**： 可以达到无口次数后进行切配吃，总开关在设置里。

## 三、 右上角设置界面

![设置界面](/images/image_51.png)

* **识别价格前延时**： 在子弹购买界面识别价格前延迟时间。
* **点击子弹前延迟**： 在收藏界面点击子弹前的延时时间。
* **无口次数**： 检测到指定次数都进入不了入手价格一下则进行切配。
* **次数达到后卡邮件，仅选择屯仓并卡邮件后生效**： 这就是主界面第三个选项勾选后要填入的次数。
* **开启切换配装**： 用过电脑端的都知道哈，就是自动切配，主界面的配装方案下边有选项，勾选后，自动达到默认检测150次无低价会切换到下一个方案，前提是自己设置好各方案参数。
* **自动检测自定义方案**： 勾选后卡邮件时会自动检测自定义1-4方案里是否可以进行卡邮件。如果不勾选，默认卡方案1里的东西

## 四、 左上角侧边栏界面

![左上角侧边栏](/images/image_52.png)

* **获取Shizuku**： 这个必须要开启，点击后会弹出授权提醒，点击确定即可

![左上角侧边栏](/images/image_53.png)

* **显示悬浮日志**： 这个必须要开启，点击后会切换到授权页面，找到软件名称，点进去打开开关即可

![左上角侧边栏](/images/image_54.png)

* **开启屏幕捕获**： 这个必须要开启，点击后会弹出授权窗口，应用范围选择【整个屏幕】然后点击开始即可。

## 五、 软件权限列表

1. 按照图片给权限，或者给全部权限。不能少给，否则可能出现异常

    ![软件权限列表](/images/image_56.png)

## 六、 使用教程

1. 侧边栏开启Shizuku，然后重启一遍软件
2. 开启显示悬浮日志、开启屏幕捕获
3. 主界面选择屯仓，设置好参数。
4. 打开游戏进入到游戏大厅界面，按音量+即可启动脚本。音量-是停止。(因为各厂商差异，这里是模拟音量键，按一下没反应可以按着等音量条动了再松开。如果在运行过程中，音量是最小则无法停止。需要按音量+把手机音量往上调一点在按音量-停止)
